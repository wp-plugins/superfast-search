The easy to use results-as-you-type search box.  Widget and shortcode.

== Description ==

Superfast Search is a search box plugin that will display your search results instantly as you type.

It comes with an easy to use Widget as well as a Shortcode.

See http://superfastsearch.co.uk/ for a guide, examples and demos.

== Installation ==

To install the plugin from your WordPress admin dashboard:

1. Plugins > Add New
2. Search for Superfastsearch
3. Install Now > Activate

To install the plugin from your computer:

1. Download the .zip file to your computer from http://superfastsearch.co.uk/
2. Upload it through Plugins > Add New > Upload > Find the zip file
3. Upload > Activate

To use the Sidebar Widget:

1. Appearance > Widgets
2. Drag the Superfast Search tab into the Sidebar panel that you want it to appear in
3. Click the Down arrow to the right of it to change the options.  Save when you are done.

To use the Shortcode:

1. Insert the line '[superfastsearch]' without quotes into the content of the page you want it to appear in
2. To see all the options you can apply to this shortcode, please visit http://superfastsearch.co.uk/
